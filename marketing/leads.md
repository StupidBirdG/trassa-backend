# Trassa — лиды для холодного охвата (растущий список)

Источник истины для агента поиска лидов (`.github/workflows/marketing-lead-finder.yml`).
Каждую неделю агент через веб-поиск ищет новые казахстанские компании (перевозчики
и грузовладельцы), которых ещё нет в этом списке, и дописывает их сюда в том же
формате. Дедупликация — по названию компании в колонке "Компания" (нижний список
"Уже найдено" ниже — просто плоский список имён для быстрой сверки).

**Это только исследование, не контакт.** Владелец сам решает, кому и как писать/звонить.
